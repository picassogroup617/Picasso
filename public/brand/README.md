# Brand Assets

Drop your logo files here. Suggested file names (the app will look for these in later phases):

- `logo.svg` — primary logo (preferred vector)
- `logo.png` — fallback raster
- `logo-mark.svg` — icon-only mark (for favicons, mobile nav)
- `og-image.png` — 1200×630 social share image

Update `NEXT_PUBLIC_BRAND_TAGLINE` in `.env.local` for your tagline.
